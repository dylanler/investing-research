# AI Latent Alpha Dashboard

Open `index.html` in a browser. The dashboard is fully static and self-contained; no internet access or local web server is required for the interactive charts and tables.

## Files

- `index.html` — main dashboard
- `assets/data/companies.csv` — ranked company data
- `assets/data/companies.json` — same data in JSON
- `assets/data/source_map.json` — source map
- `assets/css/styles.css` — dashboard styles
- `assets/js/dashboard.js` — table and SVG chart rendering
- `assets/img/mind_map.svg` — mind map visualization
- `research_methodology.md` — methodology and caveats
- `sources.md` — source list

## Important note

This is a thematic research screen, not investment advice. Re-check valuation, listing status, liquidity, estimates, backlog, and risk factors before making any decision.
