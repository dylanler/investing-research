# Strict Latent AI Alpha Dashboard

Open `index.html` in a browser. The dashboard ranks 50 U.S. and 50 non-U.S. public companies whose capabilities could become useful to AI infrastructure but that were screened to avoid current obvious AI supply-chain exposure.

Key files:
- assets/data/companies_strict_latent.csv
- assets/data/excluded_direct_ai_supply_chain.csv
- assets/data/source_pack.csv
- assets/docs/methodology.md
- assets/images/*.png
